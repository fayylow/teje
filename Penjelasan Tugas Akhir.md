Aplikasi ini merupakan program python yang digunakan untuk membuat invoice penjualan barang. 
Program invoice ini mampu menginput data dan tanggal pembeli, pembelian beberapa barang sekaligus, menhitung total harga semua dan menampilkan hasil invoice.

ALUR APLIKASI
1. Input data pembelian
2. Pembeli memasukkan jumlah barang yang mau di beli
3. Pembeli memasukkan kode barang dan berapa banyak pembelian barang tersebut
4. Program menghitung jumlah perbarang
5. Program menghitung semua total pembelian
6. Program menghitung pajak
7. Program Menghitung diskon
8. Program menampilkan semua total yang harus dibayarkan
